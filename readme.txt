-------------------------------------------------------
    Features
-------------------------------------------------------

Chosen includes the following features:

    * Custom logo
    * Social media icons
    * Show full posts or experts
    * Custom excerpt length
    * Make first post on blog extra wide
    * Custom CSS

Chosen also has a premium plugin upgrade called Chosen Pro that includes the following features:

    * Custom colors
    * Featured sliders
    * Featured videos
    * Additional layouts
    * Flexible header image
    * Google fonts
    * More widget areas
    * Display controls
    * Custom footer text

Chosen Pro: https://www.competethemes.com/chosen-pro/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Chosen can have 1-3 tiers
    * Featured Images have a 2x1 ratio at all screen sizes
    * Chosen is child theme compatible. Download a starter child theme here:
        https://www.competethemes.com/help/child-theme-chosen/
